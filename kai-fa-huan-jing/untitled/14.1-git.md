# 14.1 git

## 常用git命令

* git status
* git diff
* git add .
* git checkout ./filename 反悔之前的修改
* git commit -m "xxx"
* git log
* git show commitID
* git push origin xxx
* git pull originxxx
* git branch
* git checkout -b xxx / git checkout xxx 前往分支
* git fetch 把当前所有分支拉下来
* git merge xxx
* git stash 忘记开新分支 进行了修改 先git stash暂存最近的修改
* git stash pop 进入新分支后开放最近的修改
